
## To Run Locally
git clone https://github.com/blackrexdl/Hospital-management-working-project.git

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd Hospital-Management-System
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


# Hospital Management System

Hospital management system is a website which can control all the management related to a hospital, like admitting patients, booking beds, calling an ambulance, managing payments, creating reports of patients and many more thing online.



## Tech Stack

**Client:** 

- **React**
- **Redux Thunk**
- **Axios**
- **Ant-Designs**

**Server:**

- **Node Js**
- **Mongo DB**
- **Express Js**
- **JWT**
- **Nodemailer**
